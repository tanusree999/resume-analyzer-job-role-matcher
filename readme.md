# Resume Analyzer & Job Role Matcher

## 📌 Project Overview
This project analyzes resumes and suggests job roles using Machine Learning. It is built with Python, Streamlit, and scikit-learn.

## ⚙️ Tools Used
- Python  
- Scikit-learn  
- TF-IDF  
- SVC model  
- Streamlit  
- jupyter notebook
